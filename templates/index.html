Metodo de Euler / Euler Mejorado / Runge Kutta

Metodo de Euler: Procedimiento de integracion numerica para resolver ecuaciones diferenciales ordinarias (EDO) a partir de un valor inicial dado.

Metodo de Euler Mejorado: Consiste en tomar las formulas del metodo de Euler para calcular la pendiente en un punto inicial y en un punto final y luego promediarlas. De esta manera el resultado sera mucho mas acertado a lo largo de todo el intervalo.

Metodo de Runge Kutta: Es un conjunto de metodos iterativos (implicito y explicitos) para la aproximacion de soluciones de ecuaciones diferenciales ordinarias, concretamente, del problema de valor inicial.
